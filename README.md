# Basic Calculator

A straightforward calculator application capable of performing everyday arithmetic operations.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Clear function
- Responsive interface

## Technologies Used

- HTML
- CSS
- JavaScript

## Getting Started

### Clone the repository

```bash
git clone https://github.com/jayperkins150/Basic-calculator.git
```

### Run locally

Open `index.html` in your browser.
